# Screenshots for Landing Page

## How to Add Screenshots

1. **Take screenshots** of your app:
   - Dashboard view
   - Applications page
   - CV editor
   - Settings page
   - Any other key features

2. **Save them here** with these names:
   - `dashboard.png` - Dashboard overview
   - `applications.png` - Applications list/tracker
   - `cv-editor.png` - CV editor interface
   - `cv-preview.png` - CV preview/template
   - `experience-library.png` - Experience library
   - `settings.png` - Settings page

3. **Recommended image specs:**
   - Format: PNG or JPG
   - Width: 1200-1600px (responsive)
   - Quality: High (not too compressed)
   - Clean: Hide any sensitive test data

## Example File Structure

```
public/screenshots/
├── README.md (this file)
├── dashboard.png
├── applications.png
├── cv-editor.png
├── cv-preview.png
├── experience-library.png
└── settings.png
```
